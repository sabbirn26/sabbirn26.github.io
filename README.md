# Sabbir Nasir Portfolio

A Vite + React portfolio for an iOS software engineer with production fintech and cross-platform application experience.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

Pushes to `main` deploy through the GitHub Pages workflow in `.github/workflows/deploy.yml`.

Project screenshots live in `assets/projects/`. Recommendation quotes are managed in `src/data.js`.
